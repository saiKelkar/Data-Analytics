Select the right data:
- How the data will be collected ( data that you collect yourself is called first-part data )
- Data sources ( Second-party data is collected directly by another group and then sold, Third-party data is sold by a provider that didn't collect the data themselves )

Discrete data - that is counted and has a limited number of values.
Continuous data - that is measured and can have almost any numeric value.

Nominal data - a type of qualitative data that is categorized without a set order.
Ordinal data - a type of qualitative data with a set order or scale.

Rows ( also called as Records )
Columns ( also called as Fields )

Wide data - data in which every data subject has a single row with multiple columns to hold the values of various attributes of the subject.
Long data - data in which each row is one time point per subject, so each subject will have data in multiple rows.
